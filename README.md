# BodyTrack 📊

Eine Android-App zur Dokumentation und Visualisierung von
Körpermesswerten.

## Features
- Gewicht, Körperfett, Wasseranteil, Muskelanteil und
  Knochenmasse tracken
- Grafische Darstellung als Gesamt- oder Einzeldiagramme
- Automatische Umrechnung zwischen kg und %
- Entwicklungsstatistiken pro Zeitraum
- Drei Farbthemen: Forest, Midnight, Sunset
- CSV Export/Import
- Erweiterbar für neue Messwerte

## Tech Stack
- Kotlin + Jetpack Compose
- Material 3
- Room Database
- DataStore Preferences
- Vico Charts
- MVVM Architektur
